## Project Name & Pitch

Framer-Animations

A training project used for learning framer animations in a react peoject.


## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`
